# Big Integer Library in C++

A C++ implementation of an arbitrary-precision integer (Big Integer) library capable of performing arithmetic operations on integers larger than the limits of built-in data types.

## Features

- Addition of large integers
- Subtraction of large integers
- Multiplication of large integers
- Handles integers of arbitrary length
- Efficient string-based arithmetic implementation

## Technologies Used

- C++
- Object-Oriented Programming
- Standard Template Library (STL)

## Getting Started

### Clone the repository

```bash
git clone https://github.com/sriramjaswanth/BIG-INT.git
```

### Compile

```bash
g++ bigint.cpp -o bigint
```

### Run

```bash
./bigint
```
